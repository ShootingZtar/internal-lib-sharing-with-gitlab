## Prerequisites

Before starting, you should have:

- Basic knowledge of **Vue 3** (Composition API)
- Familiarity with **TypeScript**
- Access to a **GitLab** instance (gitlab.com or self-hosted)
- **Node.js 18+** installed on your machine
- Basic understanding of **npm** package management

## Course Structure

### Part 1: Building Your Library

Learn how to set up a Vue component library project from scratch, create reusable components, and configure the build process.

### Part 2: Publishing to GitLab

Configure your project for GitLab's npm registry and learn the publishing workflow, including versioning strategies.

### Part 3: Using the Library

Learn how to install and use your published packages in other projects within your organization.

### Part 4: Advanced Topics

Explore CI/CD automation, monorepo setups, and documentation generation.

## Why GitLab Package Registry?

| Feature | Benefit |
|---------|---------|
| **Private by default** | Keep your internal components secure |
| **Integrated with GitLab** | Use existing authentication and permissions |
| **No external dependencies** | No need for paid npm org or separate registry |
| **CI/CD ready** | Seamless integration with GitLab pipelines |
